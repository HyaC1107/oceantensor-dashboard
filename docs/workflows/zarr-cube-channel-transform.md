## 목적 (초안 원문)

Zarr ML 데이터 큐브의 채널을 재수집 없이 슬라이스 드롭, 재정렬, 수학적 역변환 및 순환 계절성 계산을 통해 새 스키마로 가공.

## 관련은 있으나 정확히 일치하지 않는 기존 코드

- `ml/data/channel_builder.py` — 원본 채널 **생성**(수집→보간→저장) 로직. 후처리 변환은 아님.
- `scripts/merge_cubes.py` — 큐브 병합 스크립트. 채널 드롭/재정렬과는 다른 작업.
